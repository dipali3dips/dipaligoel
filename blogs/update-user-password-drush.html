Drush tool in Drupal8

Written by Dipali

Posted on 2020 Jan 14

What is Drush

Drush is a command-line interface that helps us to speed up administrative and development tasks for Drupal sites. ... Note: Drupal 8, works only with Drush 8

How to Install: Drupal 8 as it requires Drush > 8. Composer is easiest method to install Drush 

$ curl -sS https://getcomposer.org/installer | php
$ mv composer.phar /usr/local/bin/composer

Now add Composer's directory to your path by editing your .bash_profile

$ export PATH="$HOME/.composer/vendor/bin:$PATH"

Now that we have Composer working, we can install the latest Drush. 

$ composer global require drush/drush:dev-master

Keep Drush updated

$ composer global update

Check if drush is successfully installed

$ drush
